Software design principles: tensions & synergies dojo
======================

Exercise based on the [Conway's Game of Life](http://en.wikipedia.org/wiki/Conway%27s_Game_of_Life)

To install dependencies:
```
> bun install
```

To run all tests:
```
> bun run test
```

To run the application:
```
> bun run dev
```

To build for production:
```
> bun run build
```
